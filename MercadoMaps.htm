Consideraciones

Página web realizada enteramente con HTML, CSS, Javascript y jQuery.

Problemas enfrentados con API de Mercadolibre V. 0.0.6:

Mercadolibre no cuenta con búsqueda a través de barrios,(Filtros disponibles son éstos) Siendo ésto asi, si los nombres de los barrios no aparecen como es debido, actualmente no están soportados por el buscador. La búsqueda realizada en la parte superior de la página se hace para toda la capital, quedando guardados en memoria todos los ítems de la búsqueda por página, separado por cada uno de los barrios.

Problemas enfrentados con API de Google Maps V. 3:
  • Los globos de información no soportan espacios en los nombres, es por ello que les puse guiones bajos.
  • Asignar un tamaño por porcentaje al mapa (width, heigth) resulta complicado, ya que para que funcione se tuvo que agregar a los estilos los campos: html { height: 100% } body { height: 100%; margin: 0px; padding: 0px } y asignar un div conteniendo al mapa, el cual debía también tener sus respectivos width y heigth también al 100%
